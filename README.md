💎**使用指南**💎

1. 首先你需要一个 [百度AI开放平台](https://ai.baidu.com/) 的账号。

2. 找到 [图片翻译](https://ai.baidu.com/tech/mt/pic_trans) 功能，点击**立即使用**进入控制台。

3. 创建一个图片翻译应用。你可以通过实名认证以领取一些免费额度（2021/11/26）。

4. 创建好应用后你会获取到其`APP_ID`和`APP_KEY`，将这两项内容填写到`run.py`中此处:

   ```python
   APP_ID = ''
   APP_KEY = ''
   ```

   

5. 运行`run.py`即可。
